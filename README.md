### 如何安装

安装必须的npm安装包： `npm install`

下载必须的前端插件： `bower install`

运行node程序： `node ./bin/www`


### 程序实现

基于`Express`、`LeanCloud`、`Semantic-UI`实现一个简易的诗词系统。在这里，将你的豪情释放、诗意舒展，不出门，一样可以坐拥诗和远方。
